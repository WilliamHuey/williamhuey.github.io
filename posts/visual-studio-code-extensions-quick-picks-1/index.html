Visual Studio Code Extensions - Quick Picks #1

06/10/2021, Thu
Categories: #editor

Active Document Indicator

Better discern the active editor group by using highlight emphasis on the current editor group while dimming inactive editor groups. This helps with accessibility even though it might only be a small visual change, it is effective in providing more focused editing.

lTeX

lTeX is a spelling and grammar checker for LaTeX and other markup language such as BibTEX, markdown, Org, resStructuredText and R Sweave using LanguageTool. Help with improving sentences readability.

reStructuredText

Live preview for reStructuredText document edits enables quick feedback when interacting with rst files.
